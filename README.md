## Description

** The Pong Game is 2 players game. In this game who ever gets 15 points is the winner. 

## Technologies Used
* HTML
* JavaScript
* CSS
* Babel
* ESLint
* Webpack
* Visual Studio Code for Mac



## Keys

**Player 1:**
* a: up
* z: down
* x: left
* s: right

**Player 2:**
* ▲ : up
* ▼: down
* ▶: right
* ◀: left

**Extra:**
* space bar: pause
* b: running the second ball
